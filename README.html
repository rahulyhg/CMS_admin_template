CMS Admin Template

A static front-end for content management system admin template

Features:

  • Bootstrap Table Classes
  • Bootswatch
  • Table Sorter
  • Glyphicons
  • Post Forms
  • Search Filer Plugin
  • Dropdown Menus
  • Breadcrumbs
  • Button Groups

Learning Highlights:

  • Build an admin UI
  • Style HTML tables
  • Add breadcrumbs and pagination
  • Sort tables via JavaScript
  • Filter Tables
